The Three Stages of Website Creation

As someone browsing my website, you might be curious how websites are created and put online.

There are three main stages of website creation:

  • Domain Registration
    Websites require a "domain" to connect to, like www.example.com or www.potato.com. Since everyone in the entire world is going to be able to connect to this website, the domain name needs to be uniquely registered in a public database that everyone can access. The domain uniquely points to a single given website. Some popular domain providers include GoDaddy and Domain.com, but many online website creator tools can automatically provide you with a website domain/subdomain too.
  • Server Hosting
    The content of any given website needs to be publicly available for people to view it. It is the job of a web server to "serve" the content of a website to anyone who connects to the website. Popular web server providers include Amazon AWS and GitHub Pages. GitHub Pages is what I used to host this website!
  • Website Development
    Finally, websites need content to actually display. This can either be raw html/css/js code created by a programmer directly (like with this website) or content developed by a WYSIWYG editor like Adobe Dreamweaver CC. I used Visual Studio Code to write the code for this website.

So there you have it. I'm glossing over a HUGE amount of detail here, but this is simple enough of an explanation for now. Website development is one of the most useful skills you can learn, so hopefully this webpage helped you learn at least some more about it!
