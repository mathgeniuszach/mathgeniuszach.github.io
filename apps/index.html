Projects

Over time I (Zach K) have put together various projects for various purposes, but mostly to learn and solve various problems I've had in the past. Below are a list of the more significant ones.

I also have a bunch of unfinished and minor stuff scattered throughout my computer, but I won't include those here.

Online Apps

Apps that you can use, right in your browser!

  • Origin Creator
    Origin/Power creator for the Origins mod in Minecraft, in addition to a generic online datapack editor.
  • Madad Maps
    Live covid bed availablility in India (I worked on scraping sources but not the website itself).

Programs

Your standard executables.

  • asset-builder A tool for choosing, recolorizing, and stacking images together of the same type.
  • fpcurator
    A bulk tool for doing many common things as a curator for Flashpoint.
  • zdpack
    A tool for merging and converting Minecraft data and resource packs.
  • The Maze Caves
    A Java based maze engine and game made for a college class.
  • TableToBingosync
    A tool that can turn copy-and-paste tables into bingosync json boards. Probably should be turned into an online tool.

Scripts and Libraries

Code that isn't a program you can run alone, but useful nonetheless

  • xdialog A cross-platform wrapper for native dialogs written entirely in python.
  • qfile
    A python library for simplifying common file operations.
  • UnifyTags
    Minecraft item unifier powered by KubeJS.
  • Trigger the Music
    Situational music data and resource pack for Minecraft.
  • fpclib
    Python library for curating games for Flashpoint.

Cool Stuff

This is some cool stuff that I don't own but think you should check out!

  • Flashpoint
    Flash and webgame preservation project. I helped out with this project quite a bit a while ago.
  • PixelVision8
    Open Source Fantasy Console that's a whole lot nicer than Pico8. I helped out a bit with a few bugs.
  • Minecraft
    The classic block game.
  • Hytale
    Upcoming (not released) block game of awesomeness. I hope it replaces Minecraft.
  • Godot
    Open Source game engine that I like better than Unity.
  • AutoHotkey
    Tool for running macro-scripts.
  • Visual Studio Code
    The IDE I used to build this website. :)
  • Parcel
    Blazing fast, zero configuration web application bundler I used to make this website. It's like magic being able to import a typescript or sass file from html.
